# 🚁 DJI Mavic 4 Pro - Landing Page

A static product landing page for the DJI Mavic 4 Pro drone, built using HTML5 and CSS3.

## 🔗 Live Demo
👉 https://kareem-alboghdadi-3.github.io/drone-product-landing-page/

## 🧾 Description
This project is a landing page that showcases the DJI Mavic 4 Pro with different sections including:

- Hero section with product highlights
- Pricing and key features
- Product comparison table
- Customer testimonials
- Accessories section
- Footer with links and social media

## 🛠️ Tech Stack
- HTML5
- CSS3

## ⚙️ Features
- Fixed navigation bar
- Smooth scrolling between sections
- Custom CSS utility classes
- Float-based layout system

## 📌 Note
This project is currently **not fully responsive** and is optimized for desktop view only.
